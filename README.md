## 🚀 快速开始：三步煮龙虾
### 1. 准备食材
```bash
# 克隆项目
git clone https://github.com/evcgs/clawcooking.git
cd clawcooking
```

### 2. 处理食材
```bash
# 一键安装依赖和环境
./scripts/install.sh

# 填写配置（所有配置需要自行填写，无预置内容）
# ① 在 recipes/base/channel.yaml 中填写7个飞书应用的appId和appSecret
# ② 在 recipes/base/model.yaml 中填写你的大模型API密钥
# ③ 在 recipes/base/agent.yaml 中填写你的用户ID
```

### 3. 出锅享用
```bash
# 一键启动所有服务
./scripts/start.sh
```

✅ 完成！现在你可以在飞书中直接和AI助手对话了。

## 📦 扩展功能
### 用户自定义技能安装
项目支持一键安装本地开发的用户自定义技能，复用你的创新能力：
```bash
# 安装用户自定义技能
./scripts/skill-install.sh <用户本地技能包路径>
```
详细说明请查看：[技能包管理文档](docs/skills.md)
> ℹ️ 安全提示：安装技能不会覆盖主Agent记忆、对话历史和现有配置，只会新增技能文件，不影响原有数据。
> 详细安全说明请查看：[安全与数据说明](docs/security.md)
