# 📮 飞书全家桶调料包

## 配置说明
```yaml
name: "飞书全家桶"
description: "全量飞书能力整合包"
version: "1.0.0"
author: "Claw Cooking"

# 应用配置
apps:
  # 主应用（通用能力）
  main:
    app_id: "cli_a9111c82cef8dbce"
    app_secret: "your-main-app-secret"
  
  # 文档专用应用（可选，有则填）
  doc:
    app_id: "cli_a92e72365878dcca"
    app_secret: "your-doc-secret"
  
  # 日历专用应用（可选，有则填）
  calendar:
    app_id: "cli_a92e72dd16b8dcc7"
    app_secret: "your-calendar-secret"

# 能力开关
capabilities:
  document: true
  sheet: true
  bitable: true
  message: true
  calendar: true
  contact: true
  drive: true
  meeting: true

# 权限配置
permissions:
  # 允许的操作范围
  allowed_operations:
    - "read"
    - "write"
    - "edit"
    - "create"
  # 高风险操作需要确认
  confirm_required:
    - "delete"
    - "batch_delete"
    - "permission_change"
    - "mass_message"
```

## 最佳实践
1. **权限最小化**：根据实际需求开启需要的能力，不需要的能力可以关闭，降低安全风险
2. **专用应用分离**：文档和日历操作建议使用专用应用，和主应用分离，避免权限冲突
3. **操作审计**：所有飞书操作都会记录日志，方便审计和追溯
4. **批量操作提醒**：批量操作前会自动提示确认，避免误操作
5. **限速保护**：自动限制API调用频率，避免触发飞书接口限流

## 常见问题
### Q：需要开通哪些飞书权限？
A：根据你需要使用的能力，在飞书开放平台开通对应的权限即可，配置文件中有明确的权限列表。

### Q：多个应用会冲突吗？
A：不会，系统会自动根据操作类型路由到对应的应用，文档操作使用Doc应用，日程操作使用Calendar应用，其他操作使用主应用。

### Q：支持企业自建应用和商店应用吗？
A：都支持，只需要填入对应的app_id和app_secret即可。

### Q：可以对接多个飞书租户吗？
A：支持，配置多个租户信息即可，自动路由到对应的租户。
