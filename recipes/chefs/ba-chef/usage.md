# BA大厨 使用指南
## 安装方法
### 方法1：一键部署（推荐）
```bash
# 在clawcook根目录运行
./scripts/deploy.sh --only ba-chef
```

### 方法2：手动安装
```bash
# 安装智能体
openclaw agents install ba-chef --config recipes/chefs/ba-chef/config.yaml

# 验证安装
openclaw agents list | grep ba-chef
```

## 配置自定义
### 1. 添加行业知识库
编辑 `config.yaml` 中的 `knowledge_base.paths` 字段，添加你的行业知识库路径：
```yaml
knowledge_base:
  paths:
    - "~/.openclaw/workspace/knowledge/你的行业"
    - "~/.openclaw/workspace/knowledge/竞品分析"
```
BA大厨会自动检索知识库中的内容，生成更贴合行业的方案。

### 2. 自定义方案模板
编辑 `config.yaml` 中的 `workflows` 字段，自定义解决方案模板：
```yaml
workflows:
  solution_design:
    output_template: |
      # 你的自定义解决方案模板
      ## 一、客户背景
      ## 二、现状痛点
      ## 三、建设目标
      ## 四、解决方案
      ## 五、实施计划
      ## 六、预算报价
      ## 七、服务支持
```

### 3. 调整分析维度
编辑 `config.yaml` 中的 `capabilities.business` 字段，开启/关闭特定的分析能力：
```yaml
capabilities:
  business:
    competitor_analysis: false  # 关闭竞品分析
    market_research: true       # 开启市场调研
```

## 使用技巧
### 技巧1：快速生成解决方案
```
@BA大厨 设计一份[行业]解决方案：
- 客户类型：[客户规模/行业]
- 预算范围：[X]万-[X]万
- 核心需求：[核心痛点/目标]
- 预期收益：[期望达成的效果]
```
BA大厨会自动生成完整的解决方案，包含架构、功能、实施路径、ROI测算。

### 技巧2：PRD需求文档生成
```
@BA大厨 写一份[产品名称]PRD：
- 产品定位：[产品定位]
- 目标用户：[用户群体]
- 核心功能：[列出核心功能点]
- 上线时间：[预期上线时间]
```
BA大厨会生成规范的PRD文档，包含功能需求、非功能需求、验收标准。

### 技巧3：需求分析和痛点挖掘
```
@BA大厨 分析这个需求：
- 需求背景：[背景描述]
- 用户反馈：[列出用户反馈]
- 现有问题：[当前存在的问题]
```
BA大厨会梳理核心痛点，给出优先级排序和解决方案建议。

### 技巧4：ROI测算和商业分析
```
@BA大厨 测算这个项目的ROI：
- 一次性投入：[X]万
- 年运维成本：[X]万
- 预期收益：[列出预期收益项]
- 项目周期：[X]年
```
BA大厨会做详细的成本效益分析，给出投资回收期和ROI指标。

## 常见问题
### Q：BA大厨可以对接我的知识库吗？
A：可以，在配置文件中添加知识库路径，BA大厨会自动检索知识库内容，生成的方案会更贴合你的业务场景。

### Q：可以生成不同行业的解决方案吗？
A：可以，只需要在指令中说明行业类型，BA大厨会根据行业特点生成对应的方案，也可以导入行业知识库优化效果。

### Q：生成的方案可以直接用吗？
A：BA大厨生成的是基础框架，你可以根据具体客户情况调整细节，尤其是报价和实施周期部分，建议根据实际情况修改。

### Q：支持生成PPT格式的方案吗？
A：默认生成Markdown格式，可以配合Doc大厨自动转换为PPT，也可以复制到Word/PPT中直接使用。

## 最佳实践
1. **方案输出前**：先给BA大厨输入客户的背景信息、痛点、预算等关键信息，信息越详细，方案越精准
2. **多方案输出**：让BA大厨生成高/中/低三个档次的方案，给客户多个选择，提高成单率
3. **ROI重点突出**：所有方案都要突出投入产出比，用数据说话，客户更容易买单
4. **案例支撑**：在知识库中添加你的成功案例，BA大厨会自动把案例融入方案中，说服力更强
5. **定期更新知识库**：定期更新行业报告、竞品信息、成功案例，让BA大厨的方案更与时俱进
