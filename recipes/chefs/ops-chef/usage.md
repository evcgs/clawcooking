# Ops大厨 使用指南
## 安装方法
### 方法1：一键部署（推荐）
```bash
# 在ccg根目录运行
./scripts/deploy.sh --only ops-chef
```

### 方法2：手动安装
```bash
# 安装智能体
openclaw agents install ops-chef --config recipes/chefs/ops-chef/config.yaml

# 验证安装
openclaw agents list | grep ops-chef
```

## 配置自定义
### 1. 调整工作时间
编辑 `config.yaml` 中的 `capabilities.calendar` 字段，设置你的工作时间：
```yaml
capabilities:
  calendar:
    work_time: ["09:30", "18:30"]  # 修改为你的工作时间
    timezone: "Asia/Shanghai"      # 修改为你的时区
    default_reminder: 30           # 提前30分钟提醒
```

### 2. 添加自动化任务
编辑 `config.yaml` 中的 `automation_tasks` 字段，添加自定义自动化任务：
```yaml
automation_tasks:
  your_custom_task:
    schedule: "0 9 * * 1"  # 每周一9点
    action: "send_weekly_reminder"  # 自定义动作
    enabled: true
```

### 3. 配置飞书日历应用
如果你有专用的Calendar飞书应用，在飞书渠道配置中添加：
```yaml
feishu:
  sub_apps:
    calendar:
      app_id: "cli_a92e72dd16b8dcc7"
      app_secret: "your-calendar-secret"
```
Ops大厨会自动使用专用的飞书应用处理日历操作，避免和主应用冲突。

## 使用技巧
### 技巧1：快速安排会议
```
@Ops大厨 安排会议：
- 主题：[会议主题]
- 参会人：[张三、李四、王五]
- 时间：[下周一下午/最近3天内]
- 时长：[2小时]
- 需要：[预定会议室/发送会议材料/自动生成纪要]
```
Ops大厨会自动检查参会人时间冲突，选择最合适的时间，完成所有安排。

### 技巧2：日程优化
```
@Ops大厨 优化我的日程：
- 时间段：[下周/下个月]
- 需求：[留出2天整块时间写方案/减少会议时间/集中安排会议]
- 优先级：[不调整重要会议/尽量不改动已确定的安排]
```
Ops大厨会给出优化方案，你确认后自动调整日程。

### 技巧3：会议纪要整理
```
@Ops大厨 整理会议纪要：
- 录音转写：[粘贴文本/飞书文档链接]
- 重点提取：[行动项/决策/待跟进事项]
- 输出：[标准会议纪要/仅行动项列表/待办清单]
- 同步：[发送给参会人/同步到项目管理工具]
```
Ops大厨会自动整理成规范的会议纪要，提取行动项并分配责任人。

### 技巧4：自动化任务设置
```
@Ops大厨 设置自动化任务：
- 名称：[任务名称]
- 触发条件：[每天18:00/每周一9点/收到特定邮件时]
- 执行动作：[生成工作日报/发送提醒/备份数据]
- 通知：[执行前确认/执行后通知/静默执行]
```
Ops大厨会创建对应的自动化任务，按设定条件自动执行。

## 常见问题
### Q：Ops大厨可以接入其他日历吗？
A：默认支持飞书日历，后续可以扩展支持Google Calendar、Outlook Calendar等，需要在配置中添加对应的权限。

### Q：可以和其他工具联动吗？
A：可以，支持和飞书项目、Jira、Trello等项目管理工具联动，自动同步行动项和任务状态。

### Q：自动化任务可以自定义吗？
A：完全可以，你可以定义任意触发条件和执行动作，支持复杂的工作流编排。

### Q：如何处理隐私问题？
A：所有日程和会议数据都在本地处理，不会上传到任何第三方服务器，你可以配置敏感会议不被索引和处理。

## 最佳实践
1. **统一入口**：所有日程安排、会议协调、任务提醒都通过Ops大厨处理，避免多入口混乱
2. **规则先行**：提前配置好工作时间、会议规则、提醒规则，Ops大厨会自动按照规则处理
3. **逐步自动化**：从简单的提醒、备份任务开始，逐步增加自动化任务，避免一开始太复杂
4. **定期回顾**：每周让Ops大厨帮你分析日程使用情况，优化时间分配
5. **敏感信息保护**：重要会议和日程可以标记为敏感，Ops大厨会特殊处理，不会泄露相关信息
