# 🧑🍳 六大厨师团队
我们基于你现有的4个核心智能体（PM/Code/Doc/Calendar），扩展为6大智能体，覆盖全场景工作需求：

| 厨师 | 状态 | 基于原有能力 | 核心能力 |
|------|------|--------------|----------|
| 👔 PM大厨 | ✅ 已有 | 原有PM智能体 | 项目管理、任务拆解、进度跟踪、风险预警 |
| 📊 BA大厨 | 🆕 新增 | 原有PM+Doc能力扩展 | 业务分析、需求挖掘、方案设计、ROI测算 |
| 📈 Data大厨 | 🆕 新增 | 原有Code+Doc能力扩展 | 数据处理、统计分析、可视化、SQL查询 |
| 👨💻 Code大厨 | ✅ 已有 | 原有Code智能体 | 代码编写、调试、审查、重构、工具开发 |
| 📝 Doc大厨 | ✅ 已有 | 原有Doc智能体 | 文档编写、格式转换、内容提取、摘要生成 |
| ⚙️ Ops大厨 | ✅ 已有 | 原有Calendar智能体扩展 | 日程管理、邮件处理、会议纪要、自动化工作流 |

## 扩展说明
1. **BA大厨**：基于你现有的PM和Doc智能体的能力，新增业务分析、方案设计、ROI测算等能力，不需要重新开发，只需要优化系统提示词和配置
2. **Data大厨**：基于你现有的Code和Doc智能体的能力，新增数据分析、可视化、SQL查询等能力，不需要重新开发
3. **Ops大厨**：基于你现有的Calendar智能体的能力，扩展邮件处理、会议纪要、自动化工作流等能力，不需要重新开发

这样既完全复用了你已有的开发成果，又丰富了智能体矩阵，形成完整的能力覆盖，不需要额外开发太多新功能，快速实现产品化。
