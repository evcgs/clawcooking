# 效能提升套餐
> Ops大厨 + PM大厨 + Doc大厨 协同工作，完成日常事务自动化处理

## 适用场景
- 日程管理和会议安排
- 邮件处理和自动回复
- 会议纪要和行动项整理
- 周报/月报自动生成
- 日常事务自动化处理
- 系统巡检和安全检查

## 工作流
```
用户需求 → Ops大厨（日程管理、邮件处理、事务自动化）
          → PM大厨（任务优先级排序、时间管理、行动项跟踪）
          → Doc大厨（纪要整理、报告生成、格式优化）
          → 用户确认
```

## 使用示例
### 示例1：会议纪要自动生成
**用户输入**：
> 这是今天的会议录音转写文本，帮我整理成会议纪要，提取行动项

**执行过程**：
1. Ops大厨自动识别参会人、会议主题、讨论内容
2. 提取核心结论、决策事项、行动项（责任人、截止时间）
3. PM大厨对行动项按优先级排序，关联到相关项目
4. Doc大厨整理成规范的会议纪要，自动发送给参会人
5. 行动项自动同步到项目管理工具，跟踪执行进度

### 示例2：周报自动生成
**用户输入**：
> 帮我生成这周的工作周报，基于我的日程、邮件、任务完成情况

**执行过程**：
1. Ops大厨自动收集本周日程、已完成任务、邮件、会议记录
2. 整理本周完成的工作、进展、问题、下周计划
3. PM大厨对工作内容进行分类，统计工作时长、产出、风险
4. Doc大厨生成规范的周报，自动发送给上级和相关人员

### 示例3：日程智能管理
**用户输入**：
> 帮我安排下周的日程，优先安排项目评审会议，留出2天整块时间写方案

**执行过程**：
1. Ops大厨扫描现有日程，找出可用时间段
2. 按照优先级安排会议，避开冲突，留出整块专注时间
3. PM大厨对任务进行优先级排序，合理分配时间
4. 自动发送会议邀请，同步日程到日历
5. 提前提醒参会人，自动准备会议材料

### 示例4：日常事务自动化
**用户输入**：
> 帮我设置一个自动化流程，每天下班前自动汇总当天工作，生成工作日报

**执行过程**：
1. Ops大厨创建自动化工作流，每天18:00自动触发
2. 自动收集当天的任务完成情况、邮件、会议、沟通记录
3. Doc大厨整理成工作日报，发送给用户确认
4. 自动同步到企业OA系统，无需手动填写

## 常用指令
```
# 会议纪要整理
@Ops大厨 这是会议录音转写，帮我整理成会议纪要，提取行动项

# 周报生成
@Ops大厨 帮我生成这周的工作周报，包含完成的工作、问题、下周计划

# 日程安排
@Ops大厨 帮我安排下周的日程，优先安排[重要事项]，留出[X]时间专注工作

# 邮件处理
@Ops大厨 帮我处理收件箱，整理出需要回复的重要邮件，草拟回复

# 自动化流程
@Ops大厨 帮我创建一个自动化流程，每天[时间]自动[执行的任务]
```

## 配置说明
- 默认支持的工具：飞书日历、飞书文档、邮箱、OA系统、项目管理工具
- 默认自动化触发：定时触发、事件触发、手动触发
- 可以自定义工作流，实现任意日常事务的自动化处理
- 可以在Ops大厨的配置文件中自定义集成工具、自动化规则、通知方式
