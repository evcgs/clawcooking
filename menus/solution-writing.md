# 方案撰写套餐
> BA大厨 + Doc大厨 + Ops大厨 协同工作，完成解决方案和文档撰写

## 适用场景
- 客户解决方案编制
- 需求文档和产品PRD
- 项目建议书和投标文件
- 业务流程优化方案
- 行业研究和分析报告
- 公众号文章和内容创作

## 工作流
```
用户需求 → BA大厨（需求分析、方案设计、痛点挖掘）
          → Doc大厨（文档撰写、结构优化、格式排版）
          → Ops大厨（竞品分析、数据支撑、素材收集）
          → 用户确认
```

## 使用示例
### 示例1：客户解决方案编制
**用户输入**：
> 帮我写一份企业数字化转型解决方案，面向制造行业客户，预算500万

**执行过程**：
1. BA大厨分析制造行业痛点、客户需求、核心价值、ROI测算
2. 设计整体解决方案架构、功能模块、实施路径、技术选型
3. Ops大厨收集行业案例、竞品分析、数据支撑、成功客户案例
4. Doc大厨整理成完整的解决方案文档，包含封面、目录、正文、附录
5. 输出可直接用于客户汇报的完整方案，支持导出为Word/PPT/PDF

### 示例2：需求文档编写
**用户输入**：
> 帮我写一个智能客服系统的需求文档PRD

**执行过程**：
1. BA大厨梳理产品定位、核心功能、用户故事、业务流程、交互逻辑
2. 定义功能需求、非功能需求、验收标准、优先级排序
3. Doc大厨按照PRD模板整理成规范的需求文档，包含原型说明、流程图
4. 输出完整的PRD文档，可直接用于研发团队评审

### 示例3：公众号文章创作
**用户输入**：
> 帮我写一篇关于AI大模型在企业应用的公众号文章，5000字左右

**执行过程**：
1. BA大厨梳理文章结构、核心观点、案例支撑、数据引用
2. Ops大厨收集行业数据、最新资讯、相关案例、图片素材
3. Doc大厨撰写文章内容，优化标题、小标题、行文逻辑、金句提炼
4. 输出排版完成的公众号文章，包含封面图建议、摘要、关键词

## 常用指令
```
# 解决方案撰写
@BA大厨 帮我写一份[行业] [产品]解决方案，面向[客户类型]，预算[X]万

# 需求文档编写
@BA大厨 帮我写一个[产品名称]的PRD需求文档，包含核心功能和业务流程

# 行业研究报告
@BA大厨 帮我做一份[行业]的研究报告，分析当前现状、未来趋势、机会点

# 公众号文章
@BA大厨 帮我写一篇关于[主题]的公众号文章，字数[X]字，风格活泼/专业

# 投标文件
@BA大厨 帮我写一份[项目名称]的投标文件技术部分，包含方案、实施计划、资质
```

## 配置说明
- 默认文档模板：解决方案、PRD、行业报告、公众号文章等10+常用模板
- 默认输出格式：Markdown、Word、PPT、PDF
- 默认字数：方案类1-2万字，PRD类5000-1万字，文章类2000-5000字
- 可以在BA大厨的配置文件中自定义行业知识库、方案模板、输出格式
